## 项目概述

### 项目简介

本项目实现了一个基于深度学习的手术视频质量评估模型，能够从手术视频片段中评估外科医生的技能水平。模型架构基于论文 paper1231，但进行了适配以支持单视频直接评分。

### 核心特性

- **双流特征提取**: 结合静态特征(ResNet-34)和动态特征(I3D)
- **掩膜引导注意力**: 使用SAM3生成的器械掩膜聚焦关键区域
- **单视频评分**: 直接回归预测技能分数，无需对比学习
- **论文对齐**: 严格遵循paper1231的架构和超参数设置

### 与原论文的区别

| 方面 | paper1231 | 本模型 |
|-------|-----------|---------|
| 框架 | 查询/参考视频对对比学习 | 单视频直接回归 |
| 交叉注意力 | 是(查询和参考之间) | 否 |
| 分数预测 | y_q = y_r + CR(E(V_q), E(V_r)) | y = R(Concat(F_static, F_dynamic)) |
| 训练目标 | 学习相对分数差异 | 学习绝对分数 |
| SAM版本 | SAM2在线生成 | SAM3离线预计算 |

### 1. 视频文件要求

**格式**: `.mp4` (推荐) 或 `.avi`, `.mov`

**编码**: H.264/H.265

**分辨率**: 建议 640x480 或更高(会自动调整到112x112)

**帧率**: 建议 25-30 FPS

**时长**: 建议 30秒+ (至少16帧)

**文件命名**:
- 必须与annotations.json中的key一致
- 推荐格式: `video_XXX.mp4` 或 `case_XXX.mp4`

**示例**:
```bash

```

### 2. 掩膜文件要求

#### 掩膜生成方式(三选一)

**方式1: 使用SAM3在线生成** (不推荐，慢)

```python
from sam3 import build_sam3_video_model
import cv2

# 初始化SAM3
model = build_sam3_video_model(device='cuda')
inference_state = model.init_state('video_path.mp4')

# 添加点提示(需要在器械上点击)
model.add_points_prompts(
    inference_state,
    frame_idx=0,
    point_coords=[[x, y]],  # 器械上的坐标
    point_labels=[1]         # 1=正点, 0=负点
)

# 传播掩膜
masks = model.propagate_in_video(inference_state)

# 保存掩膜
output_dir = 'masks/video_001'
os.makedirs(output_dir, exist_ok=True)
for frame_idx, mask in enumerate(masks):
    cv2.imwrite(f'{output_dir}/frame_{frame_idx:04d}_mask.png', mask)
```

**方式2: 使用SAM3离线生成** (推荐)

```bash
# 使用外部工具批量生成掩膜
# 示例使用SAM3命令行工具(需自行实现脚本)
```

**方式3: 使用预生成的掩膜** (最推荐)

从论文作者或其他来源获取预计算好的掩膜。

#### 掩膜文件格式

**文件命名格式**:
- 必须: `*_mask.png`
- 推荐: `frame_XXXX_mask.png` (XXXX为帧号，4位数字)
- 示例: `frame_0000_mask.png`, `frame_0001_mask.png`

**像素值**:
- 灰度图像 (单通道)
- 值范围: [0, 255] (0=背景, 255=器械)
- 程序会自动归一化到 [0, 1]

**数量**:
- 至少与视频帧数相同
- 可以更多(会被截断)
- 可以更少(会用0填充)

**示例**:
```
masks/video_001/
├── frame_0000_mask.png
├── frame_0001_mask.png
├── frame_0002_mask.png
└── ... (与视频帧数匹配)
```

#### 验证掩膜质量

```python

import numpy as np
```


### 4. 数据集划分

程序会自动按以下比例划分:

- **训练集**: 80%
- **验证集**: 10%
- **测试集**: 10%

**重要**: 划分基于 clips 数量，而非视频数！

每个视频会被提取多个 clips:
- 片段长度: 16帧
- 片段步长: 10帧
- 示例: 120帧视频 → (120-16)/10 + 1 ≈ 11个clips

### 5. 数据集验证脚本

```python
#!/usr/bin/env python
"""数据集验证脚本"""

import os
import json
import cv2
import numpy as np

data_root = '/path/to/dataset'

# 1. 检查目录结构
print("=" * 60)
print("检查目录结构")
print("=" * 60)

dirs_to_check = ['videos', 'masks', 'annotations.json']
for item in dirs_to_check:
    path = os.path.join(data_root, item)
    if os.path.exists(path):
        print(f"✓ {item}")
    else:
        print(f"✗ {item} - NOT FOUND!")

# 2. 加载标注
print("\n" + "=" * 60)
print("加载标注文件")
print("=" * 60)

anno_path = os.path.join(data_root, 'annotations.json')
with open(anno_path, 'r') as f:
    annotations = json.load(f)

print(f"视频总数: {len(annotations)}")

# 3. 检查视频文件
print("\n" + "=" * 60)
print("检查视频文件")
print("=" * 60)

video_dir = os.path.join(data_root, 'videos')
video_files = [f for f in os.listdir(video_dir) if f.endswith('.mp4')]

print(f"视频文件数: {len(video_files)}")

missing_videos = []
for video_id in annotations.keys():
    video_path = os.path.join(video_dir, f'{video_id}.mp4')
    if not os.path.exists(video_path):
        missing_videos.append(video_id)

if missing_videos:
    print(f"✗ 缺失视频: {len(missing_videos)}")
    for vid in missing_videos[:5]:
        print(f"  - {vid}")
else:
    print("✓ 所有视频文件都存在")

# 4. 检查掩膜文件
print("\n" + "=" * 60)
print("检查掩膜文件")
print("=" * 60)

mask_dir = os.path.join(data_root, 'masks')
missing_masks = []
for video_id in annotations.keys():
    video_mask_dir = os.path.join(mask_dir, video_id)
    if not os.path.exists(video_mask_dir):
        missing_masks.append(video_id)
        continue

    mask_files = [f for f in os.listdir(video_mask_dir) if 'mask' in f]
    if len(mask_files) == 0:
        missing_masks.append(video_id)

if missing_masks:
    print(f"✗ 缺失掩膜: {len(missing_masks)}")
    for vid in missing_masks[:5]:
        print(f"  - {vid}")
else:
    print("✓ 所有掩膜文件都存在")

# 5. 验证视频和掩膜匹配
print("\n" + "=" * 60)
print("验证视频-掩膜匹配")
print("=" * 60)

for video_id in list(annotations.keys())[:3]:  # 检查前3个
    # 检查视频
    video_path = os.path.join(video_dir, f'{video_id}.mp4')
    cap = cv2.VideoCapture(video_path)
    video_frames = int(cap.get(cv2.CAP_PROP_FRAME_COUNT))
    cap.release()

    # 检查掩膜
    video_mask_dir = os.path.join(mask_dir, video_id)
    mask_files = sorted([f for f in os.listdir(video_mask_dir) if 'mask' in f])
    mask_count = len(mask_files)

    print(f"{video_id}: 视频帧={video_frames}, 掩膜数={mask_count}")

print("\n✓ 验证完成!")
```

### 关键配置参数详解

#### 数据设置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `data_root` | str | 必需设置 | 数据集根目录 |
| `batch_size` | int | 8 | 批次大小 |
| `num_workers` | int | 4 | 数据加载工作进程数 |

**注意事项**:
- `data_root` 必须正确设置，否则无法加载数据
- `batch_size` 根据显存调整:
  - 6GB显存: 建议 2-4
  - 12GB显存: 建议 6-8

## 模型架构详解

### 整体架构

```
输入视频 (B, C, T, H, W)
    ├───┬─→ ResNet-34 (静态特征提取器) ──→ 静态特征 A (B, 512)
    │   │
    │   ├─ ResNet 主干网络
    │   ├─ 多尺度下采样 (s=2, 4, 8)
    │   └─ 帧采样策略 (中间帧/平均/首尾)
    │
    ├─→ I3D (动态特征提取器) ──→ 动态特征 C (B, 1024, T', H', W')
    │   ├─ 混合 3D 卷积
    │   ├─ Inception 模块
    │   └─ Mixed_5c 输出
    │
    └─→ SAM3 掩膜 (离线) ──→ 器械掩膜 B (B, T, H, W)
        └─ 从预计算文件读取

掩膜引导注意力模块
    ├─ 输入: B (掩膜) 和 C (动态特征)
    ├─ 时间平滑: 减少掩膜抖动
    ├─ 空间对齐: 匹配特征分辨率
    ├─ 生成注意力: A = mask_smoothed
    └─ 应用注意力: F_dy = C * (A + 1.0)
        └─ 输出: D (B, 1024)

特征融合与回归
    ├─ 输入: A (512维) + D (1024维) = 1536维
    ├─ 全连接层: 1536 → 1024 → 512 → 256 → 128 → 64 → 1
    └─ 输出: 分数 y (B, 1)
```

### 各组件详解

#### 1. StaticFeatureExtractor (静态特征提取器)

**文件**: `models/static_feature_extractor.py`

**功能**: 提取组织状态和手术视野的静态特征

**架构**:
```
视频输入 (B, 3, T, 112, 112)
    ├─ 帧采样 → 单帧或多帧
    ├─ ResNet-34 Backbone
    │   ├─ Conv1 (3→64)
    │   ├─ Layer1 (64→64)
    │   ├─ Layer2 (64→128)
    │   ├─ Layer3 (128→256)
    │   └─ Layer4 (256→512)
    ├─ 多尺度下采样 (MultiScaleDownsample)
    │   ├─ Scale2: 512→128, stride=2
    │   ├─ Scale4: 512→128, stride=4
    │   └─ Scale8: 512→128, stride=8
    └─ 投影 → 384 → 512
```

**输出**: 512维静态特征向量

**关键参数**:
- `output_dim`: 512 (论文要求)
- `sampling_strategy`: 'middle' (默认), 'average', 'first', 'last'
- `freeze_early_layers`: True (只训练Layer4)

#### 2. DynamicFeatureExtractor (动态特征提取器)

**文件**: `models/dynamic_feature_extractor.py`

**功能**: 提取器械操作的时空特征

**架构** (标准I3D):
```
视频输入 (B, 3, 16, 112, 112)
    ├─ Stem
    │   ├─ Conv3d_1a_7x7 (3→64, 7x7x7 kernel)
    │   ├─ MaxPool (1x3x3)
    │   ├─ Conv3d_2b_1x1 (64→64)
    │   ├─ Conv3d_2c_3x3 (64→192)
    │   └─ MaxPool (1x3x3)
    │
    ├─ Mixed_3
    │   ├─ Mixed_3b (192→256)
    │   ├─ Mixed_3c (256→256)
    │   └─ MaxPool (3x3x3)
    │
    ├─ Mixed_4
    │   ├─ Mixed_4b (384→480)
    │   ├─ Mixed_4c (544→480)
    │   ├─ Mixed_4d (480→480)
    │   ├─ Mixed_4e (576→576)
    │   ├─ Mixed_4f (576→832)
    │   └─ MaxPool (2x2x2)
    │
    ├─ Mixed_5
    │   ├─ Mixed_5b (832→832)
    │   └─ Mixed_5c (832→1024)
    │
    └─ Mixed Conv (可选)
        └─ 扩展时间感受野

输出: (B, 1024, 2, 4, 4) → 池化 → (B, 1024)
```

**输出**: 1024维动态特征向量

**关键参数**:
- `output_dim`: 1024 (标准I3D Mixed_5c输出)
- `freeze_backbone`: True (冻结I3D权重)
- `i3d_path`: 预训练权重路径

#### 3. MaskGuidedAttention (掩膜引导注意力)

**文件**: `models/mask_guided_attention.py`

**功能**: 使用器械掩膜聚焦动态特征

**流程**:
```
输入: 动态特征 C (B, 1024, T, H, W) + 掩膜 B (B, T_mask, H_mask, W_mask)
    ├─ 时间平滑 (Temporal Smoothing)
    │   ├─ 相邻帧求和: M = M^{2t-1} + M^{2t}
    │   └─ 2D平均池化: M = avg_pool2d(M, kernel=2)
    │
    ├─ 空间对齐 (Spatial Alignment)
    │   └─ 三线性插值到 I3D 特征分辨率
    │
    ├─ 生成注意力 (Attention Generation)
    │   └─ A = mask_aligned  # 直接使用掩膜作为注意力
    │
    ├─ 应用注意力 (Apply Attention)
    │   └─ F_dy = C * (A + 1.0)  # +1保留背景信息
    │
    └─ 全局池化 (Global Pooling)
        └─ adaptive_avg_pool3d → (B, 1024)
```

**输出**: 1024维掩膜动态特征

**关键特点**:
- 离线SAM3掩膜直接作为注意力(不学习)
- 时间平滑减少单帧掩膜抖动
- A + 1.0 确保背景也保留(权重≥1)

#### 4. FusionRegressor (融合回归器)

**文件**: `models/surgical_qa_model.py`

**功能**: 融合静态和动态特征，预测分数

**架构**:
```
输入: Concat(静态特征, 动态特征) → (B, 1536)
    ├─ FC: 1536 → 1024 + LayerNorm + ReLU + Dropout(0.3)
    ├─ FC: 1024 → 512  + LayerNorm + ReLU + Dropout(0.3)
    ├─ FC: 512  → 256  + LayerNorm + ReLU + Dropout(0.3)
    ├─ FC: 256  → 128  + LayerNorm + ReLU + Dropout(0.3)
    ├─ FC: 128  → 64   + LayerNorm + ReLU + Dropout(0.2)
    └─ FC: 64   → 1

输出: 分数 (B, 1)
```

**初始化**: Xavier uniform初始化

**关键参数**:
- `regressor_hidden_dims`: [1024, 512, 256, 128]

### 模型参数统计

| 模块 | 可训练参数 | 总参数 |
|------|-----------|--------|
| ResNet-34 (冻结) | ~0 | 21M |
| I3D (冻结) | ~0 | 10M |
| ResNet-34 Layer4 | ~0.5M | 0.5M |
| MaskGuidedAttention | 0 | 0 |
| FusionRegressor | ~1M | 1M |
| **总计** | **~1.5M** | **~32M** |

### 推理脚本

创建推理脚本 `inference.py`:

```python
#!/usr/bin/env python
"""
手术视频质量评估推理脚本
"""

import torch
import numpy as np
import cv2
import os
import yaml
from models import SurgicalQAModel, build_model
from utils import SurgicalQADataLoader

def load_model(config_path, checkpoint_path):
    """加载训练好的模型"""
    # 加载配置
    with open(config_path, 'r') as f:
        config = yaml.safe_load(f)

    # 构建模型
    model = build_model(config)

    # 加载检查点
    checkpoint = torch.load(checkpoint_path, map_location='cpu')
    model.load_state_dict(checkpoint['model_state_dict'])
    model.eval()

    return model, config

def process_video(model, video_path, mask_dir, device='cuda'):
    """
    处理单个视频并预测技能分数

    Args:
        model: 训练好的模型
        video_path: 视频文件路径
        mask_dir: 掩膜目录
        device: 推理设备

    Returns:
        score: 预测分数
    """
    # 1. 加载视频
    cap = cv2.VideoCapture(video_path)
    frames = []
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
        frames.append(frame)
    cap.release()

    # 2. 提取clip (取中间16帧)
    total_frames = len(frames)
    start_idx = max(0, total_frames // 2 - 8)
    end_idx = min(total_frames, start_idx + 16)

    clip = frames[start_idx:end_idx]
    if len(clip) < 16:
        # 填充
        clip.extend([clip[-1]] * (16 - len(clip)))

    # 3. 预处理
    # Resize to 112x112
    clip_resized = [cv2.resize(f, (112, 112)) for f in clip]
    clip_np = np.stack(clip_resized, axis=0)  # (T, H, W, C)
    clip_np = clip_np.transpose(3, 0, 1, 2)  # (C, T, H, W)
    clip_tensor = torch.from_numpy(clip_np).float() / 255.0
    clip_tensor = (clip_tensor - 0.5) * 2.0  # [-1, 1]
    clip_tensor = clip_tensor.unsqueeze(0)  # (1, C, T, H, W)

    # 4. 加载掩膜
    video_id = os.path.splitext(os.path.basename(video_path))[0]
    masks = load_masks(mask_dir, video_id, clip_length=16)

    if masks is not None:
        masks_tensor = masks.unsqueeze(0)  # (1, T, H, W)
    else:
        masks_tensor = None

    # 5. 推理
    with torch.no_grad():
        clip_tensor = clip_tensor.to(device)
        if masks_tensor is not None:
            masks_tensor = masks_tensor.to(device)

        model = model.to(device)
        score, _ = model(clip_tensor, masks_tensor)

    return score.item()

def load_masks(mask_dir, video_id, clip_length=16):
    """加载掩膜"""
    video_mask_dir = os.path.join(mask_dir, video_id)
    if not os.path.exists(video_mask_dir):
        return None

    mask_files = sorted([f for f in os.listdir(video_mask_dir)
                       if 'mask' in f and f.endswith('.png')])

    masks = []
    for i in range(min(clip_length, len(mask_files))):
        mask = cv2.imread(os.path.join(video_mask_dir, mask_files[i]),
                         cv2.IMREAD_GRAYSCALE)
        mask = cv2.resize(mask, (112, 112))
        mask = mask / 255.0
        masks.append(mask)

    if len(masks) > 0:
        masks_np = np.stack(masks, axis=0)
        return torch.from_numpy(masks_np).float()
    else:
        return None

def main():
    import argparse

    parser = argparse.ArgumentParser(description='Surgical QA Inference')
    parser.add_argument('--config', type=str, default='configs/default.yaml')
```

### 批量推理

```python
#!/usr/bin/env python
"""
批量推理脚本
"""

import os
import json
import torch
import yaml
from models import build_model
from tqdm import tqdm

def batch_inference(model, video_dir, mask_dir, config, device='cuda'):
    """
    批量推理

    Args:
        model: 训练好的模型
        video_dir: 视频目录
        mask_dir: 掩膜目录
        config: 配置字典
        device: 设备

    Returns:
        results: 字典 {video_id: score}
    """
    results = {}

    # 获取所有视频
    video_files = [f for f in os.listdir(video_dir) if f.endswith('.mp4')]

    # 推理
    for video_file in tqdm(video_files, desc="Inference"):
        video_id = os.path.splitext(video_file)[0]
        video_path = os.path.join(video_dir, video_file)

        # 处理视频
        score = process_video(model, video_path, mask_dir, device)

        results[video_id] = {
            'score': score,
            'video_file': video_file
        }

    return results

def main():
    # 解析参数
    parser = argparse.ArgumentParser()
    parser.add_argument('--checkpoint', required=True)
    parser.add_argument('--video_dir', required=True)
    parser.add_argument('--mask_dir', required=True)
    parser.add_argument('--output', default='predictions.json')
    args = parser.parse_args()

    # 加载模型
    with open('configs/default.yaml', 'r') as f:
        config = yaml.safe_load(f)

    model = build_model(config)
    checkpoint = torch.load(args.checkpoint, map_location='cpu')
    model.load_state_dict(checkpoint['model_state_dict'])
    model.eval()

    # 批量推理
    results = batch_inference(
        model,
        args.video_dir,
        args.mask_dir,
        config
    )

    # 保存结果
    with open(args.output, 'w') as f:
        json.dump(results, f, indent=2)

    print(f"Results saved to {args.output}")

if __name__ == '__main__':
    main()
```

```python
"""

import torch

def extract_features(model, video_tensor, mask_tensor):
    """
    提取中间特征

    Args:
        model: SurgicalQAModel
        video_tensor: (1, C, T, H, W)
        mask_tensor: (1, T, H, W)

    Returns:
        features: 字典包含各层特征
    """
    with torch.no_grad():
        score, features = model(
            video_tensor,
            mask_tensor,
            return_features=True,
            return_attention=True
        )

    return features

def visualize_attention(attention_map, original_frame, save_path):
    """
    可视化注意力图

    Args:
        attention_map: (T, H, W)
        original_frame: (H, W, C)
        save_path: 保存路径
    """
    # 选择中间帧
    mid_frame = attention_map.shape[0] // 2
    attention = attention_map[mid_frame]

    # 归一化到[0, 255]
    attention_vis = (attention * 255).astype(np.uint8)

    # 创建热力图
    heatmap = cv2.applyColorMap(attention_vis, cv2.COLORMAP_JET)

    # 叠加到原图
    frame_bgr = cv2.cvtColor(original_frame, cv2.COLOR_RGB2BGR)
    blended = cv2.addWeighted(frame_bgr, 0.7, heatmap, 0.3, 0)

    # 保存
    cv2.imwrite(save_path, blended)

def main():
    # 加载模型
    with open('configs/default.yaml', 'r') as f:
        config = yaml.safe_load(f)

    model = build_model(config)
    checkpoint = torch.load('best_checkpoint.pth', map_location='cpu')

    original_frame = load_frame('video.mp4')
```

### 模型剪枝

```python
#!/usr/bin/env python
"""
模型剪枝以减小模型大小
"""

import torch.nn.utils.prune as prune

def prune_model(model):
    """剪枝模型"""
    # 剪枝融合回归器的线性层
    for module in model.fusion_regressor.regressor:
        if isinstance(module, torch.nn.Linear):
            # L1非结构化剪枝
            prune.l1_unstructured(module, name='weight', amount=0.2)

    # 移除剪枝掩膜(永久剪枝)
    for module in model.fusion_regressor.regressor:
        if isinstance(module, torch.nn.Linear):
            prune.remove(module, 'weight')

    print("Model pruned")
    model.count_parameters()

def main():
    model = build_model(config)
    checkpoint = torch.load('checkpoint.pth')
    model.load_state_dict(checkpoint['model_state_dict'])

    # 剪枝前
    print("Before pruning:")
    model.count_parameters()

    # 剪枝
    prune_model(model)

    # 保存剪枝后的模型
    torch.save(model.state_dict(), 'pruned_model.pth')
```

**解决方案**:

## 最佳实践

### 1. 数据准备

- **视频质量**: 使用高质量、稳定的手术视频
- **标注一致性**: 确保所有评分使用相同标准
- **掩膜准确性**: SAM3掩膜应准确覆盖器械
- **数据量**: 最少50个视频，推荐100+

### 2. 训练策略

- **学习率**: 从小开始，逐步增加
- **Early Stopping**: 必须启用，防止过拟合
- **验证集**: 确保验证集具有代表性
- **检查点**: 定期保存，防止意外中断

### 3. 模型部署

- **模型大小**: 剪枝或量化减小模型
- **推理速度**: 使用TensorRT加速
- **批处理**: 批量推理提高吞吐量

### 4. 评估与对比

- **多次运行**: 随机种子不同，运行多次取平均
- **指标全面**: 不仅看MAE，也要看SRCC、PCC
- **可视化**: 分析注意力图和特征响应
