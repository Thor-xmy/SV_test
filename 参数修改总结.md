```python
def build_optimizer(model, config):
    # 论文要求：
    # - Backbone (I3D3D + ResNet): 1e-4
    # - Static module (fusion regressor): 1e-3
    # - Weight decay: 0

    # 分组参数
    backbone_params = []  # static_extractor + dynamic_extractor
    other_params = []      # fusion_regressor等

    for name, param in model.named_parameters():
        if 'static_extractor' in name or 'dynamic_extractor' in name:
            backbone_params.append(param)
        else:
            other_params.append(param)

    # 构建参数组
    param_groups = [
        {'params': backbone_params, 'lr': 1e-4, 'name': 'backbone'},
        {'params': other_params, 'lr': 1e-3, 'name': 'static_module'}
    ]

    optimizer = optim.Adam(param_groups, weight_decay=0.0)
```

**代码位置**：`train.py` 第179-232行

**修改位置**：`__getitem__` 方法（第362-432行）

**关键改进的地方**：
1. **统一的增强决策**：在加载frames之前决定是否翻转/旋转
2. **一致的增强应用**：将相同的增强应用于frames和masks

```python
def __getitem__(self, idx):
    # 1. 决定增强变换（对frames和masks一致）
    apply_horizontal_flip = False

    if self.is_train:

        if hasattr(self, 'enable_rotation') and self.enable_rotation:
            if random.random() < 0.5:
                rotation_k = random.choice([1, 2, 3])  # 90, 180, 270度

    # 2. 预处理frames（应用增强）

    # 3. 加载masks
    # ... 加载、resize、slice步骤 ...

    # 4. 应用相同的增强到masks
    if apply_horizontal_flip:
```

**为什么需要统一增强**：
- SAM3掩膜对应视频中器械的位置
- 如果frames水平翻转，masks也必须水平翻转
- 如果frames旋转，masks也必须旋转相同角度
- 否则掩膜与视频内容将不匹配，导致模型学习错误

## ⚠️ 注意事项

### 1. 分学习率的必要性

论文要求不同模块使用不同学习率：
- **Backbone** (I3D + ResNet): `1e-4` (0.0001)
  - 这些是预训练模型，使用较小学习率微调
- **Static Module** (Fusion Regressor): `1e-3` (0.001)
  - 这些是从头训练的层，使用较大学习率加速收敛

### 2. 掩膜与视频同步增强

SAM3掩膜指示视频中器械位置：
- 如果视频被水平翻转，掩膜也必须水平翻转
- 如果视频旋转90°，掩膜也必须旋转90°
- 修改已确保**帧和掩膜使用相同的随机决策**

### 3. Early Stopping Patience

论文使用patience=10，代码默认是15：
- Patience=10: 更激进的early stopping
- 适合论文中的100-150 epoch训练设置
