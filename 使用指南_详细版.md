## 模型整体架构

### 框架流程图

```
手术视频输入 X (B, C, T, H, W)
    │
    ├──> [1] 静态特征提取 (ResNet-34)
    │         │
    │         ├──> ResNet-34 Backbone
    │         │       ├──> Conv1: 7x7, stride=2, 64通道
    │         │       ├──> MaxPool: 3x3, stride=2
    │         │       ├──> Layer1: 3个残差块
    │         │       ├──> Layer2: 4个残差块
    │         │       ├──> Layer3: 6个残差块
    │         │       └──> Layer4: 3个残差块
    │         │
    │         ├──> 多尺度下采样 (Multi-Scale)
    │         │       ├──> Scale 2: Conv2d(3x3, stride=2) → 128通道
    │         │       ├──> Scale 4: Conv2d(3x3, stride=4) → 128通道
    │         │       └──> Scale 8: Conv2d(3x3, (3) → 128通道
    │         │
    │         └──> 投影层: Linear(384 → 512)
    │
    │         └──> 输出: 静态特征 A (B, 512)
    │
    ├──> [2] 动态特征提取 (I3D)
    │         │
    │         ├──> I3D Backbone
    │         │       ├──> Stem: Conv3d(1x7x7, stride=1,2) → 64通道
    │         │       ├──> Mixed_3b: 192 → 256通道
    │         │       ├──> Mixed_3c: 256 → 256通道
    │         │       ├──> Mixed_4b-4f: 256 → 480通道
    │         │       └──> Mixed_5b-5c: 480 → 832通道
    │         │
    │         ├──> Mixed Conv3D (可选)
    │         │       ├──> Temporal 3D卷积
    │         │       ├──> Spatial 3D卷积
    │         │       └──> 融合层
    │         │
    │         └──> 输出: 动态特征 C (B, 832, T', H', W')
    │
    ├──> [3] 掩膜引导注意力 (SAM3掩膜)
    │         │
    │         ├──> 输入: SAM3离线掩膜 B (B, T, H, W)
    │         │
    │         ├──> 时序平滑 (Temporal Smoothing)
    │         │       ├──> 相邻帧求和: M^(2t-1) + M^(2t)
    │         │       ├──> 2D平均池化: f_2D
    │         │       └──> 输出: (B, T/2, H/2, W/2)
    │         │
    │         ├──> 空间对齐 (Spatial Alignment)
    │         │       └──> 3D插值: 调整到I3D特征分辨率
    │         │
    │         └──> 注意力应用: F_clip ⊙ (A + I)
    │
    │         └──> 输出: 掩膜特征 D (B, 832)
    │
    ├──> [4] 特征融合与回归
    │         │
    │         ├──> 拼接: Concat(A, D) → (B, 1344)
    │         │
    │         └──> MLP回归器
    │               ├──> FC1: Linear(1344 → 1024) + LayerNorm + ReLU + Dropout(0.3)
    │               ├──> FC2: Linear(1024 → 512) + LayerNorm + ReLU + Dropout(0.3)
    │               ├──> FC3: Linear(512 → 256) + LayerNorm + ReLU + Dropout(0.3)
    │               ├──> FC4: Linear(256 → 128) + LayerNorm + ReLU + Dropout(0.3)
    │               ├──> FC5: Linear(128 → 64) + LayerNorm + ReLU + Dropout(0.2)
    │               └──> FC6: Linear(64 → 1)
    │
    └──> 输出: 质量分数 y (B, 1)
```

### 关键设计特点

1. **双流特征提取**: 同时提取静态（组织/视野）和动态（器械操作）特征
2. **掩膜引导注意力**: SAM3离线掩膜直接作用于动态特征，增强器械区域权重
3. **直接回归**: 单视频直接回归分数（不同于论文的对比学习）
4. **可训练模块**: 支持分层冻结/解冻，渐进式训练策略

## 模型各层详细说明

### 1. 静态特征提取器 (StaticFeatureExtractor)

#### ResNet-34 Backbone

| 层 | 输入 | 输出 | 参数量 | 冻结策略 |
|------|------|------|--------|---------|
| Conv1 | (B, 3, H, W) | (B, 64, H/2, W/2) | 9408 + 64 + 512 | - |
| BN1 | (B, 64, H/2, W/2) | (B, 64, H/2, W/2) | 128 + 128 | - |
| MaxPool1 | (B, 64, H/2, W/2) | (B, 64, H/4, W/4) | 0 | - |
| Layer1 (3块) | (B, 64, H/4, W/4) | (B, 64, H/4, W/4) | 约73K/块×3 | ResNet残差连接 |
| Layer2 (4块) | (B, 64, H/8, W/8) | (B, 128, H/8, W/8) | 约147K/块×4 | ResNet残差连接 |
| Layer3 (6块) | (B, 256, H/16, W/16) | (B, 256, H/16, W/16) | 约295K/块×6 | ResNet残差连接 |
| Layer4 (3块) | (B, 512, H/32, W/32) | (B, 512, H/32, W/32) | 约2.2M/块×3 | ResNet残差连接 |

**总参数量**: 约21.3M (ImageNet预训练权重)

#### 多尺度下采样模块

| 分支 | 输入 | 输出 | 卷积核 | 步长 |
|------|------|------|--------|------|
| Scale-2 | (B, 512, H', W') | (B, 128, H', W') | 3×3 | 2 |
| Scale-4 | (B, 512, H', W') | (B, 128, H', W') | 3×3 | 4 |
| Scale-8 | (B, 512, H', W') | (B, 128, H', W') | 3×3 | (不完整填充) |

**多尺度输出**: (B, 384, 1, 1) → flatten → (B, 384)

#### 投影层

| 层 | 输入 | 输出 | 激活函数 |
|------|------|------|--------|
| Linear-1 | (B, 384) | (B, 512) | ReLU |
| Linear-2 | (B, 512) | (B, 512) | ReLU |

**总输出**: (B, 512) 静态特征

### 2. 动态特征提取器 (DynamicFeatureExtractor)

#### I3D Backbone

| 层 | 输入 | 输出 | 通道数 | 参数量 |
|------|------|------|--------|--------|
| Stem Conv | (B, 3, T, H, W) | (B, 64, T, H/2, W/2) | 64 | ~9.4K |
| Stem BN | (B, 64, T, H/2, W/2) | 同上 | 64 | 128 |
| Stem Pool | (B, 64, T, H/2, W/2) | (B, 64, T, H/4, W/4) | 0 | 0 |
| Mixed_3b/c | (B, 192, ...) | (B, 256, ...) | 256 | ~300K |
| Mixed_4b-4f | (B, 256, ...) | (B, 480, ...) | 480 | ~1.5M |
| Mixed_5b/c | (B, 480, ...) | (B, 832, ...) | 832 | ~2.5M |

**I3D总参数量**: 约12.7M (Kinetics预训练权重)

#### Mixed Conv3D模块（可选）

| 层 | 输入 | 输出 | 核大小 |
|------|------|------|--------|
| Standard 3D Conv | (B, C, T, H, W) | (B, C, T, H, W) | (T, 3, 3) |
| Temporal 3D Conv | (B, C, T, H, W) | (B, C, T, H, W) | (T, 1, 1) |
| Spatial 3D Conv | (B, C, T, H, W) | (B, C, T, H, W) | (1, 3, 3) |
| Fusion Conv | (B, 2C, T, H, W) | (B, C, T, H, W) | (1, 1, 1) |

**输出**: 空间池化后 → (B, 832, T, 1, 1) → flatten → (B, 832)

### 3. 掩膜引导注意力模块 (MaskGuidedAttention)

#### 时序平滑

**公式**: M_gt[w,h] = f_2D(∑(M^(2t-1,w,h) + M^(2t,w,h))) / 2

- **输入**: (B, T, H, W) 掩膜
- **输出**: (B, T/2, H/2, W/2) 平滑后掩膜
- **作用**: 减少单帧掩膜的抖动和局部不稳定性

#### 空间对齐

- **输入**: (B, T/2, H/2, W/2) 平滑掩膜
- **输出**: (B, T', H', W') 对齐到I3D特征分辨率
- **方法**: 3D trilinear插值
- **注意**: trilinear会产生0-1之间的连续值，对深度学习友好

#### 注意力应用

**公式**: F_dy = F_clip ⊙ (A + I)

- **F_clip**: I3D输出特征 (B, 832, T', H', W')
- **A**: 对齐后的掩膜注意图 (B, T', H', W')
- **I**: 恒等1.0标量
- **效果**:
  - 器械区域 (A≈1): 权重 × 2
  - 背景区域 (A≈0): 权重 × 1

#### 全局池化

- **输入**: 掩膜后的特征 (B, 832, T', H', W')
- **输出**: (B, 832) 全局平均池化
- **作用**: 聚合成紧凑表示，用于后续融合

---

### 4. 特征融合与回归模块 (FusionRegressor)

#### MLP回归器结构

| 层 | 输入 | 输出 | 激活 | Dropout |
|------|------|------|------|--------|
| FC1 | (B, 1344) | (B, 1024) | ReLU | 0.3 |
| LayerNorm1 | (B, 1024) | (B, 1024) | - | - |
| FC2 | (B, 1024) | (B, 512) | ReLU | 0.3 |
| LayerNorm2 | (B, 512) | (B, 512) | - | - |
| FC3 | (B, 512) | (B, 256) | ReLU | 0.3 |
| LayerNorm3 | (B, 256) | (B, 256) | - | - |
| FC4 | (B, 256) | (B, 128) | ReLU | 0.3 |
| LayerNorm4 | (B, 128) | (B, 128) | - | - |
| FC5 | (B, 128) | (B, 64) | ReLU | 0.2 |
| LayerNorm5 | (B, 64) | (B, 64) | - | - |
| FC6 (输出) | (B, 64) | (B, 1) | - | - |

**总参数量**: 约2.2M (可训练）

**初始化策略**: Xavier均匀初始化

## 数据格式准备

### 1. 目录结构

```
your_dataset/
├── videos/                  # 手术视频文件
│   ├── video_001.mp4
│   ├── video_002.mp4
│   └── ...
├── masks/                   # SAM3分割掩膜
│   ├── video_001/          # 每个视频一个文件夹
│   │   ├── frame_0000_mask.png
│   │   ├── frame_0001_mask.png
│   │   └── ...
│   ├── video_002/
│   │   └── ...
└── annotations.json          # 视频标签文件
```

### 2. 视频格式要求

**格式**: MP4 (推荐) 或 AVI
**编码**: H.264 或类似
**分辨率**: 建议224x224或更高
**帧率**: 建议25-30 FPS
**长度**: 建议16帧以上（会自动分段）

**注意**: 模型会自动resize到224x224

### 3. 掩膜格式要求

**格式**: PNG (推荐) 或 JPEG
**尺寸**: 与原始视频相同（推荐）
**值域**:
- 背景: 0 (黑色)
- 器械: 255 (白色)
- 边界: 可以是过渡值

**文件命名**: `frame_{帧号:04d}_mask.png`

示例:
```
frame_0000_mask.png  # 第0帧掩膜
frame_0001_mask.png  # 第1帧掩膜
...
```

### 4. 标注格式

```json
{
    "video_001": {
        "score": 7.5,
        "duration": 120.5,
        "surgeon": "胆囊切除术",
        "phase": "Calot三角解剖"
    },
    "video_002": {
        "score": 8.2,
        "duration": 115.3,
        "surgeon": "胆囊切除术",
        "phase": "Calot三角解剖"
    }
}
```

**score**: 质量分数（必填，浮点数）
**duration**: 视频时长秒（可选）
其他字段可以自定义，但score是必需的。

### 5. 数据示例生成

使用项目提供的工具生成示例数据：

```python
from utils.data_loader import create_sample_annotations

# 生成标注文件
create_sample_annotations(
    output_dir="/path/to/your/dataset",
    num_videos=100
)
```

## 训练流程

### 1. 基本训练命令

```bash
python train.py \
    --data_root /path/to/your/dataset \
    --output_dir ./output \
    --batch_size 8 \
    --epochs 100 \
    --learning_rate 1e-4 \
    --gpus 0
```

### 2. 完整训练参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| --data_root | None | 数据集根目录（必需） |
| --output_dir | ./output | 输出目录 |
| --batch_size | 8 | 批次大小 |
| --epochs | 100 | 训练轮数 |
| --learning_rate | 1e-4 | 学习率 |
| --weight_decay | 1e-4 | 权重衰减 |
| --optimizer | adam | 优化器：adam/sgd/adamw |
| --freeze_backbone | True | 是否冻结backbone |
| --use_pretrained | True | 是否使用预训练权重 |
| --use_amp | False | 是否使用混合精度 |
| --early_stopping | True | 是否早停 |
| --patience | 15 | 早停耐心值 |
| --seed | 42 | 随机种子 |

### 3. 配置文件方式

创建 `configs/my_config.yaml`:

```yaml
# 数据配置
data_root: /path/to/your/dataset
batch_size: 8
num_workers: 4

# 模型配置
static_dim: 512
dynamic_dim: 832
use_pretrained: true
freeze_backbone: true
resnet_path: null
i3d_path: null

# 训练配置
epochs: 100
learning_rate: 0.0001
weight_decay: 0.0001
optimizer: adam

# 高级配置
use_amp: false
clip_grad_norm: 1.0
early_stopping: true
patience: 15
min_delta: 0.001
```

运行:

```bash
python train.py --config configs/my_config.yaml
```

### 4. 训练阶段策略

**阶段1: 仅训练回归器（快速收敛）**

```yaml
freeze_backbone: true  # 冻结ResNet和I3D
```

**阶段2: 解冻上层，微调**

```python
# 从阶段1的checkpoint继续训练
model.unfreeze_backbone(layers_to_unfreeze=['layer4', 'Mixed_5b', 'Mixed_5c'])
```

**阶段3: 全量微调**

```python
# 解冻所有层
model.unfreeze_backbone(layers_to_unfreeze=['all'])
```

### 5. 损失函数

**总分损失**: `Loss = Loss_score + λ_mask * Loss_mask`

- **Loss_score**: 分数回归MSE损失
- **Loss_mask**: 掩膜监督损失（可选）
- **λ_mask**: 掩膜损失权重（默认0.1）

### 6. 评估指标

| 指标 | 说明 |
|------|------|
| MAE | 平均绝对误差，越小越好 |
| SRCC | Spearman等级相关系数，越大越好 |
| PCC | Pearson相关系数，越大越好 |
| RMSE | 均方根误差，越小越好 |

## 推理流程

### 1. 单视频推理

```python
from models import build_model
import torch

# 加载模型
config = {
    'checkpoint_path': 'output/checkpoints/best_model.pth',
    'static_dim': 512,
    'dynamic_dim': 832
}
model = build_model(config)
model.eval()

# 准备输入
video = torch.randn(1, 3, 16, 224, 224)  # (B, C, T, H, W)
masks = torch.rand(1, 16, 224, 224)  # (B, T, H, W)
masks = (masks > 0.5).float()  # 二值化

# 推理
with torch.no_grad():
    score, _ = model(video, masks)
    print(f"Predicted score: {score.item():.2f}")
```

### 2. 批量视频推理

```python
from utils.data_loader import SurgicalQADataLoader

# 创建数据加载器
dataloader = SurgicalQADataLoader(
    data_root='/path/to/your/dataset',
    batch_size=1,
    num_workers=4,
    dataset_kwargs={
        'clip_length': 16,
        'clip_stride': 10,
        'spatial_size': (224, 224),
        'cache_clips': True
    }
)

# 批量推理
model.eval()
all_scores = []
all_video_ids = []

with torch.no_grad():
    for batch in dataloader.get_loader('test'):
        video = batch['frames'].to(device)
        masks = batch['masks'].to(device) if batch['masks'] is not None else None
        video_ids = batch['video_id']

        score, _ = model(video, masks)
        all_scores.extend(score.cpu().numpy().flatten())
        all_video_ids.extend(video_ids)

# 输出结果
for vid, score in zip(all_video_ids, all_scores):
    print(f"{vid}: {score:.2f}")
```

## 常见问题

### Q1: 训练时显存不足（CUDA OOM）

**原因**: 视频数据加载过多

**解决方案**:

1. 减小batch_size
```yaml
batch_size: 4  # 从8减到4
```

2. 减少数据加载线程
```yaml
num_workers: 2  # 从4减到2
```

3. 禁用clip缓存
```yaml
cache_clips: false
```

### Q2: 训练收敛很慢

**原因**: 学习率设置不当

**解决方案**:

1. 使用学习率调度器（已内置）
```yaml
learning_rate: 0.001  # 初始学习率
```

2. 分阶段训练
```python
# 阶段1：大学习率，冻结backbone
config = {'learning_rate': 0.001, 'freeze_backbone': true}

# 阶段2：小学习率，解冻部分层
config = {'learning_rate': 0.0001, 'freeze_backbone': false}
model.unfreeze_backbone(['layer4', 'Mixed_5b'])
```

### Q3: 掩理结果分数范围不对

**原因**: 标注分数范围与模型输出不匹配

**解决方案**:

1. 检查标注分数范围
```python
# 统计分数范围
import json
with open('annotations.json') as f:
    data = json.load(f)
scores = [v['score'] for v in data.values()]
print(f"Score range: [{min(scores):.1f}, {max(scores):.1f}]")
```

2. 调整MLP输出层（在配置中）
```yaml
# 如果分数在[5, 10]范围，可能需要调整最后一层
# 模型当前输出任意实数，没有sigmoid/tanh限制
```

### Q4: 掩膜加载失败

**原因**: 掩膜文件命名或格式不匹配

**解决方案**:

数据加载器支持多种格式：
- `*_mask.png` (SAM3默认)
- `*.png` (任意PNG)
- `*.jpg` (JPEG格式)

确保掩膜文件夹命名正确。

### Q5: I3D预训练权重加载失败

**原因**: Kinetics权重路径错误或格式不兼容

**解决方案**:

1. 使用项目自带的I3D（未预训练）
```yaml
use_pretrained_i3d: false
i3d_path: null
```

2. 或下载正确的Kinetics权重
```bash
# 从GitHub下载I3D RGB模型
wget https://github.com/pierluigifloco/i3d-from-kinetics/releases/download/v1/rgb_imagenet.pt
```

### Q6: 如何可视化注意力图？

**解决方案**:

```python
    video, masks,
    return_features=True,
    return_attention=True
)

attention_map = features['attention_map']  # (B, T', H', W')

# 使用项目提供的可视化工具
from models.mask_guided_attention import AttentionVisualization
AttentionVisualization.visualize_attention(
    original_video=video[0],
    attention_map=attention_map[0],
    save_path='./attention_vis'

```

### Q7: 如何使用自己的预训练权重？

**解决方案**:

对于ResNet-34:
```python
# 使用ImageNet预训练
config = {'use_pretrained': true}
```

对于I3D：
```python
# 使用Kinetics预训练

    'i3d_path': '/path/to/rgb_imagenet.pt',
    'use_pretrained_i3d': true
}
```

### Q8: 训练多轮数后过拟合

**原因**: 训练轮数过多，模型容量不足

**解决方案**：

1. 增强正则化
```yaml
weight_decay: 0.01  # 从0.0001增加到0.01
```

2. 增加Dropout率（在代码中修改）
```python
# 在models/surgical_qa_model.py的FusionRegressor中
# 将Dropout从0.3增加到0.5
```

3. 提前停止
```yaml
early_stopping: true
patience: 10  # 从15减到10
```
