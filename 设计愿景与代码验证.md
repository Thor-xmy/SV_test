# 设计愿景与代码库验证报告

## ✅ 你的设计愿景

基于SAM3和I3D的手术视频质量评估框架：

1. **输入ResNet-34** → 提取静态特征 **A**
2. **SAM3处理** → 提取手术器械掩膜 **B**
3. **输入I3D** → 提取动态特征 **C**，通过**B直接作用**于C → 得到器械前景引导的动态特征 **D**
4. **A与D拼接** → 混合特征 → **全连接层回归**视频分数

**与论文区别**：
- 论文：对比学习（Query/Reference视频对），有交叉注意力
- 你：单视频回归，无对比学习，直接拼接后回归

#### 3.2 掩膜引导注意力（关键验证！）

**你的设计**: SAM3掩膜B直接作用于I3D特征C → 得到D

**代码实现**: `models/mask_guided_attention.py`

```python
class MaskGuidedAttention(nn.Module):
    def forward(self, dynamic_features, masks):
        """
        关键公式（符合你的设计）：
        F_dy = F_clip ⊙ (A + I)

        其中：
        - F_clip: I3D动态特征 (C)
        - A: SAM3掩膜经过时空对齐后 (B)
        - I: 单位矩阵1.0
        """
        # 步骤1: 时间平滑（论文公式：M_gt[w,h] = f_2D(sum(M^{2t-1} + M^{2t})))
        masks_smoothed = self._temporal_smoothing(masks, target_T=T)

        # 步骤2: 空间对齐（匹配I3D特征分辨率）
        masks_aligned = F.interpolate(masks_smoothed, size=(T, H, W))

        # 步骤3: 应用掩膜作为注意力（你的核心设计！）
        # 公式：F_dy = F_clip ⊙ (A + I)
        # 注意力A就是SAM3掩膜B
        masked_features = dynamic_features * (masks_aligned + 1.0)

        # 步骤4: 全局池化得到紧凑表示
        # 输出: (B, 832) = D
        pooled = F.adaptive_avg_pool3d(masked_features, (1,1,1)).flatten(1)

        return pooled  # = D
```

**与论文的区别**（代码中已注释说明）：

| 方面 | 论文 (SAM2) | 你的代码 (SAM3) |
|------|-------------|----------------|
| 掩膜来源 | SAM2在线分割 | SAM3离线预计算 |
| 注意力学习 | 学习注意力A，掩膜监督 | 直接使用掩膜作为注意力 |
| 监督信号 | 有mask loss | 无mask loss（掩膜是ground truth） |

**验证结果**: ✅ **完全符合你的设计**
- SAM3掩膜直接作用于I3D特征 ✅
- 使用公式 `F_dy = F_clip ⊙ (A + I)` ✅
- 没有额外的注意力学习网络 ✅

### 4. 特征拼接与回归

**你的设计**: A与D拼接 → 混合特征 → 全连接层回归分数

**代码实现**: `models/surgical_qa_model.py`

```python
class SurgicalQAModel(nn.Module):
    def __init__(self, config):
        # 1. 静态特征提取器
        self.static_extractor = StaticFeatureExtractor(...)  # 提取A

        # 2. 动态特征提取器
        self.dynamic_extractor = DynamicFeatureExtractor(...)  # 提取C

        # 3. 掩膜引导注意力
        self.masked_attention = MaskGuidedAttention(...)  # C ⊙ B → D

        # 4. 融合回归器（全连接层）
        self.fusion_regressor = FusionRegressor(
            static_dim=512,  # A的维度
            dynamic_dim=832  # D的维度
        )

    def forward(self) -> score:
        # 步骤1: 提取静态特征A
        static_features = self.static_extractor(video)  # (B, 512) = A

        # 步骤2: 提取动态特征C
        dynamic_feat_map, _ = self.dynamic_extractor(video, return_features_map=True)  # (B, 832, T, H, W) = C

        # 步骤3: 掩膜引导得到D
        # B（掩膜）直接作用于C（I3D特征）
        masked_dynamic = self.masked_attention(dynamic_feat_map, masks)  # (B, 832) = D

        # 步骤4: 融合A和D后回归
        score = self.fusion_regressor(static_features, masked_dynamic)  # (B, 1)
        return score
```

**验证结果**: ✅ **完全符合你的设计**
- A和D拼接 ✅
- 通过全连接层MLP回归 ✅
- 无对比学习框架 ✅
- 无cross-attention ✅
